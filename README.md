# Introduction
삼성카드 데이터분석 공모전(SCDC2020)에 참여하여 진행한 프로젝트입니다. 대회 주제는 ‘온라인 가맹점 방문 고객 예측 알고리즘 개발’입니다. 빅데이터와 AI기술을 이용하여 온라인 가맹점에 방문할 고객을 예측하는 모델을 제안해야 합니다. 데이터를 이미지화 하여 공개된 이미지 모델을 Transfer Learning할 수 있도록 하였습니다.
